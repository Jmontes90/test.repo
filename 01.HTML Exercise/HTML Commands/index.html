HTML Commands

<img>
  • purpose: embeds an image into the document.
  • example: <img src="somefunnypicture.jpg" alt="some funny pictures">
  • attributes:
    • Attributes Values Meaning
    src URL source file
    height number, percentage the intrinsic height of the image in pixels. must be an interger without a unit.
    alt text Defines an alternative text description of the image

<a>, </a>
  • purpose: anchor element, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.
  • example:<a href="https://en.wikipedia.org/wiki/cat">wiki Cats.</a>
  • attributes:
    • Attributes Values Meaning
    href URL The URL that the hyperlink points to.
    target name_of_frame Where to display the linked URL, as the name for a browsing context (a tab, window <iframe>
    _blank usually a new tab, but users can configure browsers to open a new window instead.
    _parent the parent browsing context of the current one. If no parent, behaves as _self.
    _top the topmost browsing context (the "highest" context thats an ancestor of the current. If no ancestors, behaves as _self.
    _self the current browsing context.(default)

  • attributes
    • Attributes Values Meaning
    this element only includes the global attributes. Note: The aling Attribute on <p> tags is obsolete and shouldn't be used.
